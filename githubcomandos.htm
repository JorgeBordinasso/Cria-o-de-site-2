Comandos Git

  • git fetch --all

    • sincorniza o repositório local com o repositório remoto

  • git pull

    • faz o download das atualizações do servidos remoto

Finalização das atividades

  • git status

    • Listas as modificações

  • git add .

    • Adiciona mas modificações no servidor local

  • git commit -m "nome do commit"

    • nibta as modificações para envio ao servidor local

  • git push

    • faz o upload das modificaçõesdo servidor local para o servidor remoto
